# Détails du paiement

Récupère toutes les informations financières liées à un paiement, y compris le calcul des frais et les écritures comptables (Ledger). 

<Note>
  Cet endpoint est principalement utilisé pour la réconciliation et l'audit financier.
</Note>

## Path Parameters

| Paramètre | Type | Requis | Description |
|-----------|------|--------|-------------|
| `id` | string | ✅ | ID du paiement (UUID) |
