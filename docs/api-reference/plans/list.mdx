# Lister les plans

Récupère la liste de tous vos plans d'abonnement.

## Endpoint

```
GET /v1/plans
```

## Headers

| Header | Requis | Description |
|--------|--------|-------------|
| `Authorization` | ✅ | `Bearer sk_xxx` |

## Exemple

<CodeGroup>

```bash cURL
curl https://api.sahelpay.ml/v1/plans \
  -H "Authorization: Bearer sk_test_xxx"
```

```javascript Node.js
const response = await fetch('https://api.sahelpay.ml/v1/plans', {
  headers: {
    'Authorization': 'Bearer sk_test_xxx',
  },
});
```

</CodeGroup>
