## Installation rapide

<CodeGroup>

```bash JavaScript
npm install @sahelpay/sdk
# ou
yarn add @sahelpay/sdk
```

```bash PHP
composer require sahelpay/php
```

```bash Python
pip install sahelpay
```

</CodeGroup>

## Fonctionnalités

Tous les SDKs exposent les mêmes fonctionnalités :

| Fonction | Description |
|----------|-------------|
| `createPayment()` | Créer un paiement |
| `getPaymentStatus()` | Vérifier le statut |
| `verifyWebhook()` | Vérifier la signature webhook |
| `paymentLinks.create()` | Créer un lien de paiement |
